# 後中醫國文

後中醫國文備考資料整理專案，用來集中保存原始資料、整理筆記、參考來源與輸出成果。

## 資料夾

- `input/`: 原始題目、講義、PDF、截圖或待整理資料。
- `notes/`: 整理後的 Markdown 筆記與主題摘要。
- `references/`: 參考書目、來源連結、考情資料與補充材料。
- `output/`: 產出的表格、整理稿、圖片或匯出檔。
- `scripts/`: 後續若需要批次轉檔、整理或檢查資料，可放輔助腳本。

## 建議流程

1. 把原始資料放進 `input/`。
2. 依主題或年份整理到 `notes/`。
3. 需要交付或複習的版本輸出到 `output/`。
4. 重要專案狀態與下一步記在 `AGENTS.md`，方便下次接續。
